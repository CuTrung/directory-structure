#

#### Template commit:

> `action[module]: description + #issue`
>
> - action: fix, feat, refactor, chore, update, ...
> - module: module name
> - description: describe what is commit do
> - issue: issue fixing (optional)
>
>   **_VD: chore(@News): update UI #3_**
